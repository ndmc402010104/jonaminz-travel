# PRODUCT VISION

## 產品定位

Jonaminz Travel 是 Jonathan 與 Minz 共用的旅行規劃、旅途中使用、旅行書出版與回憶整理系統。

它不是：

- 一般行程表。
- 一般旅遊後台。
- Canva 的完整替代品。
- 只在旅行前使用的規劃工具。
- 只有漂亮封面的展示網站。

它要完成一條長期流程：

```text
建立旅行
→ 收集 Place 素材
→ 安排到 Day
→ 形成 Stop 行程
→ 選 Page Master
→ 生成旅行書初稿
→ 套用 Book Style
→ 手動調整
→ 旅途中使用
→ 記錄 Actual 與回憶
→ 回來整理成正式旅行書
```

## 使用情境

### 旅行前

- 收集景點、餐廳、購物、交通、住宿與備選。
- 排入每天。
- 建立結構化行程。
- 生成 A4 旅行本。
- 準備手機 Today 模式、票券與導航入口。

### 旅行中

- 顯示下一站與今日時間軸。
- 標記完成、跳過與實際變更。
- 快速開啟地圖、票券與住宿。
- 加入照片與一句回憶。
- 不破壞原始 Planned 行程。

### 旅行後

- 補照片、心得、實際到訪資料。
- 轉成完整旅行書或相簿。
- 保留可回顧、可輸出、可再利用的結構化資料。

## 設計方向

- Travel App 外框：清楚、舒服、可長時間操作。
- 旅行書：可以每一本有不同美術世界。
- 目前偏好的第一套 Book Style：精品手帳拼貼。
- 高級感來自留白、層級、材質與構圖，不來自縮小功能文字。
- Browser zoom 100% 是主要驗收尺寸。
